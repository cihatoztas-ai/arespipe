# AresPipe — Standart Kütüphane Kapsamı

> **Bu belge referanstır.** Ne zaman kütüphane tabloları kurulacak ve doldurulacak diye sorulduğunda buraya bakılır.
> 40. oturum kapanış sohbetinden çıktı, vizyon belgesinin (Vizyon 9) destekleyici belgesidir.

---

## Bu Belgenin Amacı

Vizyon belgesinin **Bölüm 2.A — Parça Kimliği Prensibi** diyor ki: *"Sisteme giren bir malzeme yazı değil, fiziksel nesne olarak tanımlansın."*

Bu prensibin çalışması için **kütüphane** lazım. Bu belge o kütüphanenin **kapsam haritasıdır**. Yapılacaklar listesi değil, **referans listesidir**.

---

## Kapsam — 7 Malzeme Grubu

AresPipe gemi sektörü odaklı olduğu için **boru, fitting, flanş** kapsamında çalışır. Vana, armatür, montaj malzemeleri kapsam dışıdır.

### 1. Karbon Çelik

**Borular:**
- ASTM A106 (SA106) — Yüksek sıcaklık dikişsiz (Grade A, B, C)
- ASTM A53 — Düşük basınç, dikişsiz veya kaynaklı
- ASTM A333 — Düşük sıcaklık servisi
- API 5L — Hat borusu (line pipe)
- EN 10216 — DIN/EN eşdeğer dikişsiz boru
- EN 10217 — DIN/EN kaynaklı boru

**Buttweld Fittings (NPS ≥ 2"):**
- ASTM A234 (WPB, WPC, WP12, WP22)
- ASTM A420 (WPL3, WPL6) — Düşük sıcaklık
- ASTM A860 (WPHY 42/52/60/65/70) — Yüksek dayanım hat borusu
- EN 10253 — DIN/EN eşdeğer

**Forged Fittings (NPS ≤ 2"):**
- ASTM A105 — Karbon çelik dövme parça
- ASTM A350 (LF1, LF2) — Düşük sıcaklık
- ASTM A182 (F1, F2, F11, F22) — Alaşımlı çelik dövme
- EN 10222 — DIN/EN eşdeğer

**Flanşlar:**
- ASTM A105 — En yaygın karbon çelik flanş
- ASTM A350 — Düşük sıcaklık flanşları
- ASTM A694 — Yüksek dayanım hat borusu flanşları
- EN 1092-1 — DIN/EN flanş

### 2. Paslanmaz Çelik (Austenitik)

**Borular:**
- ASTM A312 (TP304, TP304L, TP316, TP316L, TP321, TP347)
- ASTM A358 — Elektrik füzyon kaynaklı yüksek sıcaklık
- ASTM A778 — Genel korozif servis kaynaklı
- EN 10216-5 — DIN/EN eşdeğer

**Fittings:**
- ASTM A403 (WP304/L, WP310S, WP316/L)
- ASTM A182 (F304/L, F316/L, F321, F347) — Forged + flanş
- EN 10253-3 — DIN/EN eşdeğer

**Flanşlar:**
- ASTM A182 (F304/L, F316/L)
- EN 1092-1 — DIN/EN

### 3. Duplex ve Super Duplex

**Borular:**
- ASTM A790 — Duplex (UNS S31803/S32205) ve super duplex (UNS S32750/S32760)
- ASTM A928 — Elektrik füzyon kaynaklı duplex/super duplex

**Fittings:**
- ASTM A815 (WP-S31803, WP-S32750)

### 4. Bakır-Nikel (CuNi) — Gemi İçin Kritik

**Borular:**

- ASTM B467 — Kaynaklı CuNi
- DIN 86019 — Alman gemicilik standardı

**Fittings:**
- ASTM B122, B171, B151, B466, B467 — CuNi forged ve buttweld

- EEMUA 145, 146 — İngiliz CuNi

**Flanşlar:**
- ASTM B151 — CuNi flanş hammaddesi
- DIN 86087/86088 — Alman CuNi flanş
- EEMUA 145 — İngiliz CuNi flanş

### 5. Alüminyum

**Borular:**
- ASTM B241/B241M — Alüminyum dikişsiz boru ve ekstrüde tüp
- ASTM B210 — Çekilmiş dikişsiz tüp

### 6. Alaşımlı Çelik (Düşük/Orta Alaşım)

## Kütüphane Dışı Parçalar

Bu kapsam **standart parçaları** içerir. Aşağıdakiler **özel parçalar tablosu**na (`ozel_parcalar`) gider:

- **Coupling Sleeve** — atölye üretim veya üretici-spesifik
- **Custom Fittings** — tasarımcının (AVEVA'da) çizdiği non-standard parçalar
- **Reinforcement Sleeves** — tamir/takviye halkaları
- **Mekanik Couplings** — Victaulic, Gruvlok, Straub gibi patentli sistemler
- **Üretici-spesifik kalemler** — özel üretim, proje bazlı

Bunlar zaman içinde proje girişiyle sisteme dahil olur, kütüphane standardı içinde değildir.

## DB Şeması (Vizyon Belgesi'nden Alıntı)

### fitting_olculer

```sql
fitting_olculer (
  id                UUID PRIMARY KEY,

  -- Standart referansı
  geometri_std      TEXT,      -- B16.9, B16.11, EN10253, DIN86089...
  parca_tipi        TEXT,      -- 90LR, 90SR, 45, tee_eq, tee_red,
                                -- reducer_conc, reducer_ecc, cap,
                                -- stub_long, stub_short, return_180, 3D_elbow

  -- Boyut
  cap_buyuk_nps     TEXT,
  cap_buyuk_dn      INT,
  cap_buyuk_mm      NUMERIC,
  cap_kucuk_nps     TEXT,      -- redüksiyon/reducing tee için
  cap_kucuk_dn      INT,
  cap_kucuk_mm      NUMERIC,

  schedule          TEXT,
  et_mm             NUMERIC,

  -- Geometri (parça tipine göre dolan kolonlar)
  ucu_uca_a_mm      NUMERIC,   -- A: center-to-end (dirsek)
  ucu_uca_b_mm      NUMERIC,   -- B: center-to-end (kısa kol)
  ucu_uca_c_mm      NUMERIC,   -- C: 45° elbow
  ucu_uca_f_mm      NUMERIC,   -- F: reducer toplam uzunluk
  ucu_uca_m_mm      NUMERIC,   -- M: tee center-to-end (run)
  ucu_uca_h_mm      NUMERIC,   -- H: cap toplam uzunluk
  yaricap_mm        NUMERIC,   -- dirsek bend radius (1D, 1.5D, 3D)

  -- Stub end özel
  stub_lap_kalinlik_mm NUMERIC,
  stub_lap_od_mm       NUMERIC,

  agirlik_kg        NUMERIC,
  notlar            TEXT,
  aktif             BOOLEAN
);
```

### flansh_olculer

```sql
flansh_olculer (
  id                UUID PRIMARY KEY,

  geometri_std      TEXT,      -- B16.5, B16.47, EN1092, DIN86087
  flansh_tipi       TEXT,      -- WN, SO, SW, BL, LJ, TH
  basinc_sinifi     TEXT,      -- Class 150/300/600/900/1500/2500 veya PN6/10/16/25/40

  cap_nps           TEXT,
  cap_dn            INT,
  cap_mm            NUMERIC,

  flansh_od_mm      NUMERIC,
  flansh_kalinlik_mm NUMERIC,
  hub_od_mm         NUMERIC,
  hub_uzunluk_mm    NUMERIC,
  raised_face_od_mm NUMERIC,
  raised_face_kalinlik NUMERIC,

  bolt_circle_mm    NUMERIC,
  bolt_count        INT,
  bolt_cap_mm       NUMERIC,
  bolt_uzunluk_mm   NUMERIC,

  agirlik_kg        NUMERIC,
  notlar            TEXT,
  aktif             BOOLEAN
);
```

### malzeme_kataloglari

```sql
malzeme_kataloglari (
  id                UUID PRIMARY KEY,

  malzeme_grubu     TEXT,      -- karbon, paslanmaz, duplex, cuni, alum, nikel, alasimli
  spec_kodu         TEXT,      -- A106, A234, A312, A403, A105, B361, B466, B467
  grade             TEXT,      -- B, WPB, TP304L, F316, 5083, C70600
  uns_kodu          TEXT,      -- K03006, S30403, A95083, C70600

  din_eşdeğer       TEXT,      -- 1.0345, 1.4404, AlMg4.5Mn
  en_eşdeğer        TEXT,      -- P235GH, X2CrNiMo17-12-2

  uygun_boru        BOOLEAN,
  uygun_fitting     BOOLEAN,
  uygun_flansh      BOOLEAN,

  cekme_min_mpa     NUMERIC,
  akma_min_mpa      NUMERIC,
  sicaklik_min_c    INT,
  sicaklik_max_c    INT,
  korrozyon_jsonb   JSONB,     -- {"deniz": true, "asit": false, ...}

  notlar            TEXT
);
```

### fitting_malzeme_uyum (Çapraz)

```sql
fitting_malzeme_uyum (
  fitting_id        UUID REFERENCES fitting_olculer,
  malzeme_id        UUID REFERENCES malzeme_kataloglari,
  uretim_yontemi    TEXT,      -- seamless, welded, forged, cast
  yaygin_mi         BOOLEAN,
  PRIMARY KEY (fitting_id, malzeme_id)
);
```

### ozel_parcalar (Standart Dışı)

```sql
ozel_parcalar (
  id                UUID PRIMARY KEY,
  tenant_id         UUID,
  parca_tipi        TEXT,      -- coupling_sleeve, custom_fitting, reinforcement_sleeve
  uretici           TEXT,      -- varsa
  uretici_kodu      TEXT,
  malzeme_id        UUID REFERENCES malzeme_kataloglari,

  ic_cap_mm         NUMERIC,
  dis_cap_mm        NUMERIC,
  uzunluk_mm        NUMERIC,
  et_mm             NUMERIC,

  proje_spesifik_mi BOOLEAN,
  kaynak            TEXT,      -- uretici_katalog/cizim/atolye_uretim
  notlar            TEXT
);
```

### tenant_spec_seti (AVEVA-vari)

```sql
tenant_spec_seti (
  id                UUID PRIMARY KEY,
  tenant_id         UUID,
  spec_kodu         TEXT,      -- "CS150", "SS316_300"
  ad                TEXT,      -- "Karbon Çelik Class 150"

  boru_malzeme_default     UUID REFERENCES malzeme_kataloglari,
  fitting_malzeme_default  UUID REFERENCES malzeme_kataloglari,
  flansh_malzeme_default   UUID REFERENCES malzeme_kataloglari,

  basinc_sinifi     TEXT,      -- "Class 150" veya "PN16"
  sicaklik_aralik   TEXT,      -- "-29 to 425°C"

  notlar            TEXT
);

spec_kural (
  id                UUID PRIMARY KEY,
  spec_id           UUID REFERENCES tenant_spec_seti,
  parca_tipi        TEXT,
  izinli_olculer_jsonb  JSONB,
  varsayilan_parca_id   UUID REFERENCES fitting_olculer
);
```

## Doldurma Stratejisi (Vizyon 9 — Kademeli Kurulum)

### Aşama 1: DB Schema Migration (1 oturum)
- 5 ana tablo + ilişkiler kurulur
- Boş tablolarla pilot başlar
- Test verisi (5-10 örnek satır manuel)

### Aşama 2: Veri Kaynakları (Yasal Yollar)

**a. Üretici katalogları (en pratik):**
- Octal Pipe Fittings, Haihao, Sandvik, Vallourec, Tubacex
- Public PDF, ASME B16.9 ölçüleri birebir
- Birden fazla kataloğu çapraz kontrol = doğruluk garantisi

**b. Açık kaynak datasetler:**
- GitHub: pipe-fitting-data, pipe-data
- FreeCAD'in pipe library'si
- NIST açık verileri
- Lisans: MIT/CC, yasal

**c. Kullanıcının elindeki Excel/PDF:**
- Sektörde dolaşan public bilgi
- Tersanenin yıllar içinde topladığı tablolar
- En kaliteli kaynak (zaten doğrulanmış, kullanılan değerler)

**d. ASME standardı (opsiyonel):**
- $1500-2000 yatırım
- Tek seferlik, en kapsamlı ama lisanslı
- Üreticilerden geçtiği için **gerek yok** (üreticiler ASME ölçüsünü zaten basıyor)

### Aşama 3: AI-Asistanlı Çıkarım

1. Kaynak (PDF/Excel/HTML tablo) AI'a gösterilir
2. AI, DB şemasına uygun JSON çıkarır (50-200 satır)
3. Otomatik validation (çap-schedule mantıklı mı, ölçü pozitif mi)
4. Kullanıcı **rastgele 5 satır** kaynak ile karşılaştırır
5. Eşleşiyorsa SQL INSERT'lere dönüştürülür
6. Supabase'e import

**Bir parça tipi (örn. tüm 90LR):** ~30 dakika
**Tüm fitting kütüphanesi:** Tek oturumda biter

### Aşama 4: Spool Akışına Entegrasyon

- Lambda v4 prompt'u parametrik çıktı verecek şekilde güncellenir
- Eşleştirme logic'i: AI çıktısı → fitting_olculer.id
- Fuzzy match (yakın eşleşme önerisi)
- Eşleşmeyen parça için "kütüphaneye ekle" akışı

### Aşama 5: Yönetim UI'sı

- `fitting_kutuphane.html` sayfası (5 sekme)
- Manuel ekleme modal'ı
- Excel import/export
- "Bu kayıt N projede kullanılıyor" istatistiği

### Aşama 6: AVEVA-vari Spec Sistemi (İleri Vizyon)

- `tenant_spec_seti` ve `spec_kural` tabloları kullanılır
- Tersane bazlı parça alt kümeleri
- Yeni proje girişi otomatik filtrelenir
